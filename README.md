# firefox
Build Firefox with patches and a custom pdf.js version on Linux and macOS.

* Basic requirements:
    - python3
    - ruby
    - git-cinnabar

* pdf.js requirements
    - npm
    - gulp-cli


## Arch
```bash
paru -S git-cinnabar
sudo pacman -S wasi-compiler-rt wasi-libc wasi-libc++ wasi-libc++abi
```
